Humble Package Manager

Overview

Humble Package Manager, as the name implies, is a small package manager designed with simplicity in mind. It is able to install and remove software packages; nothing more, nothing less.

A package manager is a program designed to manage the software packages installed on a system; these include the web browser, download manager, music player and more. In fact, a package manager is responsible for managing each and every file.

Usage

Humble Package Manger (or hpm, in short) consists of two parts:

  • hpm-install, a package installer.
  • hpm-remove, a package remover used to remove the packages previously installed using the former.
